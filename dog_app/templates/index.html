About this app:

This web application holds the convolutional neural network which classifies photo of your choice to one
of the 133 labeled dog breeds. If you provide a photo of an human, the application will return the most
resembling dog breed. If you want to see more details about how this app works under the hood please click
the git hub icon on the left side of this webpage. It will redirect you to the correspondend repository.
